# CLI Calculator - Task 1
## Objective

The goal of this task was to create a calculator that can perform basic arithmetic operations using functions. The program runs continuously until the user chooses to exit.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Handles invalid input
- Menu driven interaction

## Concepts Used

- Functions
- While loop
- If-elif-else conditions
- User input using input()

## How to Run

1. Make sure Python is installed on your system.
2. Open the terminal in the project folder.
3. Run the following command:

   python calculator.py

4. Follow the instructions displayed in the terminal.

This task helped me understand how to structure a Python program and build a simple CLI based application.
